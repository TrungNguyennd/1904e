    src="https://znews-photo.zadn.vn/w480/Uploaded/neg_sfzyrwj/2019_05_06/download_30.jpg"

    https://znews-photo.zadn.vn/w480/Uploaded/neg_sfzyrwj/2019_05_06/download_30.jpg sẽ là giá
    trị của thuộc tính src

    thuộc tính style dduocj sử dụng để trang trí co thẻ html
    bên trong thẻ style sẽ chứa tên thuộc tính css và giá trị của no
    ví dụ style="color: red"
    =>color sẽ là tên thuộc tính của css chỉ màu sắc
    và red là giá trị của thuộc tính đó
    kết thúc cặp tên:giá_trị là dấu;
    thuộc tính style có thể chứa nhiều cặp giá trị thuộc tính: giá trị của css
    ví dụ style="color:red; background: green;"
    => background là thuộc tính chỉ màu nên trong css
